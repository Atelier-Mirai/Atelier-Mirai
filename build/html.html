HTML / CSS を学ぶ

対面での講座

プログラミングを習得されたいあなたへ向けて、独学でも学べるよう、多数のオンライン講座もございますが、分からない点、疑問点を気軽に聞ける対面での講座は、学習意欲の維持や、技術習得の観点から、お勧めです。

HTMLとは

アトリヱ未來では、HTML 講座 と、 Ruby on Rails による Webアプリ開発講座のご提供が可能です。HTML 講座 と Webアプリ開発講座の違いですが、HTML 講座 は、あらかじめ用意された文書を表示する普通のWebサイト(ホームページ)を作成されたい方向け。Webアプリ開発講座は、さらに一歩進んで、利用者からの入力を反映するような、動的なWebサイトの開発を行います。

費用や学習期間

HTML講座は、比較的習得が容易で、初めての方にお勧めです。学習期間ですが、週一回二時間、おおよそ二ヶ月あれば、基礎的なWebサイト(このサイトもそうです)の作成は習得可能かと思います。

学習費用ですが、概ね一回二時間で３０００円〜４０００円をみていただければと思っておりますが、ご予算に応じてご相談に応じます。

学びの形

受講形態ですが、パソコン教室というよりは、喫茶店やファミリーレストランのようなところでの、家庭教師のようなイメージを持っていただければ良いかと思いますし、お友達もご一緒にということであれば、公共施設や会議室を借りてのグループレッスンも可能です。

お近くの喫茶店でミニ講座も

もしご都合がよろしければ、喫茶店等で、お打ち合わせや、パソコンをお持ちでしたら、簡単な開発環境の整備やミニ講座も可能です。

学習内容

HTML基本

  • HTMLとは

    「見出し」「本文」など、Webサイトの文書構造を書き表すための言語です。

  • HTMLで、簡単なWebサイトを書いてみる

    <h1>こんにちは</h1>

  • CSSとは

    色や大きさ、配置など、Webサイトの飾り付けを書き表すための言語です。

  • CSSで、簡単に飾り付けをしてみる。

    h1 {  color: red;}

  • CSSグリッドレイアウト

    「グリッド」とは、「格子」のこと。
    <画面の二次元>（縦と横）方向に格子(グリッド)を用意し、自由自在に、文章や画像など、各要素を配置することができるようになります。

  • レスポンシブデザイン

    メディアクエリ を使い、
    画面の幅に応じて、iPhoneやMac等、それぞれに適したデザインで、Webサイトを見せる方法を学びます。

  • お問い合わせフォーム

    Webサイトを見た方からの問い合わせを受けられるようにサイトを作っていきます。

  • Webサイトの公開

    作ったWebサイトをみんなに見てもらえるよう、サーバに公開する方法です。

  • エディタの使い方

    HTML / CSS を書くための便利なソフト、「エディタ」の使い方も学びます。

HTML発展

  • slim

    html を より簡単に書ける slim について学びます。

  • sass

    css を より簡単に書ける sass について学びます。

  • git

    ソースコードを管理できる Git について学びます。
    写真を追加した、SNSを追加したなど、
    でき上がるたびに、Git を使って保存していくと、
    元に戻したいときにすぐに戻せるなど、いろいろ便利です。
    バックアップにもなり、また他者とのコードの共用も容易になります。

  • middleman

    静的サイトジェネレータであるmiddlemanを利用すると、
    slimやscssで書いたコードをブラウザが理解できるよう、
    htmlとcssに翻訳してくれます。
    また、メニュー項目など、Webサイトで共通する部分も容易に管理できます。
