HTML とは

ウェブサイト(ホームページ)は、「HTML」 と 「CSS」という二つの言語で作り上げていきます。
数多くのプログラミング言語がある中で、HTML / CSS は、理解も容易で大変習得しやすく、作り上げた成果物（ウェブサイト）をすぐに確認できるので、学習意欲も継続します。初めての方が学ぶにはとってもお薦めの言語です。

学習内容

HTML講座では、ウェブサイトをご自身で作成できるよう、様々なことを学んでいきます。

HTML入門篇

  • HTMLとは

    「見出し」「本文」など、ウェブサイトの「文書構造」を書き表すための言語です。

  • HTMLで簡単なウェブサイトの「見出し」を書いてみます。

    <h1>こんにちは</h1>

  • CSSとは

    色や大きさ、配置など、ウェブサイトの飾り付けを書き表すための言語です。

  • CSSで、先ほど書いた「見出し」を「赤く」飾り付けをしてみます。

    h1 {  color: red; }

  • CSSグリッドレイアウト

    「グリッド」とは「格子」のことです。画面の二次元（縦と横）方向に格子を用意することで、自由自在に文章や画像などを配置できるようになります。

  • レスポンシブデザイン

    iPhone / iPad / Mac など、ウェブサイトを見る環境は様々です。それぞれの画面幅に適したデザインで、ウェブサイトを見てもらえるように作り上げていきます。

  • お問い合わせフォーム

    ウェブサイトを見た方からの問い合わせを受け付けられるよう、サイトを作っていきます。

  • ウェブサイトの公開

    作ったウェブサイトをみんなに見てもらえるよう、サーバに公開します。

  • プログラミングのための高性能文字編集ソフト「エディタ」を使う。

    HTML / CSS を書くための便利なソフト「エディタ」の使い方も学びます。

HTML発展篇

  • JavaScript(ジャバスクリプト)

    JavaScript は、文字や絵の表示を制御したり、利用者の操作に応じて動きのあるサイトを作成する際に用いられるプログラミング言語です。HTML / CSS より、少し学習の難易度は上がりますが、魅力的なサイト作成には欠かせない技術です。ウェブサイトに Google Map や YouTube を載せたり、写真などの扱い方を学びます。

  • Slim(スリム)

    HTML を より簡単に書ける Slim について学びます。

  • Sass(サス)

    CSS を より簡単に書ける Sass について学びます。

  • Git(ギット)

    ソースコードを管理できる Git について学びます。
    文章や写真を追加するなど、ウェブサイトは日々成長していきます。制作の区切りとなるところで、Git を使って保存して行きましょう。

    • 新しい機能を試しやすい。
    • 元に戻したいときにすぐに戻せる。
    • ほかの人とコード共用できる。
    などなど、とても便利です。

  • Middleman(ミドルマン)

    静的サイトジェネレータである Middlemanを利用するとウェブサイトの作成がとても捗ります。
    メニュー項目など共通する部品を管理する、SlimやSassで書いたコードを HTML/CSS に翻訳する、同じような部品を繰り返し配置するなど、様々な利点があります。
    このサイトも Middleman を使って作成しています。
