HTMLとは

ウェブサイト(ホームページ)は、HTML/CSS という言語で作成されています。
HTML/CSS は、数あるプログラミング言語の中でも、<理解も容易で>、大変習得しやすく、成果物（つまり作成したウェブサイトそのものです)が、すぐに確認できることから、学習意欲も継続しやすいという特徴を備えており、初めて学ぶにはとっても良い言語です。

学習内容

HTML講座では、ウェブサイトをご自身で作成できるよう、様々なことを学んでいきます。

HTML入門篇

  • HTMLとは

    「見出し」「本文」など、ウェブサイトの文書構造を書き表すための言語です。

  • HTMLで、簡単なウェブサイトを書いてみる。

    <h1>こんにちは</h1>

  • CSSとは

    色や大きさ、配置など、ウェブサイトの飾り付けを書き表すための言語です。

  • CSSで、簡単に飾り付けをしてみる。

    h1 {  color: red; }

  • CSSグリッドレイアウト

    「グリッド」とは、「格子」のこと。画面の二次元（縦と横）方向に格子を用意することで、自由自在に文章や画像などを配置する方法を学びます。

  • レスポンシブデザイン

    iPhone / iPad / Mac など、ウェブサイトを見る環境は様々です。それぞれの画面幅に適したデザインで、ウェブサイトを見せる方法を学びます。

  • お問い合わせフォーム

    ウェブサイトを見た方からの問い合わせを受けられるよう、サイトを作っていきます。

  • ウェブサイトの公開

    作ったウェブサイトをみんなに見てもらえるよう、サーバに公開する方法です。

  • エディタの使い方

    HTML / CSS を書くための便利なソフト、「エディタ」の使い方も学びます。

HTML発展篇

  • JavaScript(ジャバスクリプト)

    JavaScript は、文字や写真の表示を制御したり、利用者の操作に応じて動きのあるサイトを作成する際に用いられるプログラミング言語です。html / css に比べ、学習の難易度は上がりますが、魅力的なサイト作成には欠かせない技術です。Google Map や YouTube、写真などの扱い方を学びます。

  • Slim(スリム)

    HTML を より簡単に書ける Slim について学びます。

  • Sass(サス)

    CSS を より簡単に書ける Sass について学びます。

  • Git(ギット)

    ソースコードを管理できる Git について学びます。
    文章や写真を追加するなど、ウェブサイトは日々成長していきます。制作の区切りとなるところで、Git を使って保存して行きましょう。新しい機能を試しやすい、元に戻したいときにすぐに戻せる、バックアップにもなる、他者とのコード共用もできるなど、とても便利です。

  • Middleman(ミドルマン)

    静的サイトジェネレータである Middlemanを利用すると、メニュー項目など、ウェブサイトで共通する部分も容易に管理できる、SlimやSassで書いたコードをブラウザが理解できるよう、HTML / CSS に翻訳できるなど、様々な利点があります。
