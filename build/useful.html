HTML / CSS

ウェブサイトを作成するために用いられる言語で、HTMLは文書構造を、CSSは文書装飾を担当します。

  • ウェブ入門 - MDN web docs

    HTML (HyperText Markup Language; ハイパーテキストマークアップ言語) はウェブページのコンテンツを記述や定義するための言語です。 詳細で分かり易く使用法が記載されているリファレンスサイトです。

  • HTML - MDN web docs

    完全な初心者へ向けて、実用的なウェブ開発の入門シリーズです。まずはじめに、簡単なウェブページを作るのに必要なツールを導入するところから始めて、簡単なコードを書いたら、実際にウェブに公開するところまでを道案内します。

  • CSS - MDN web docs

    CSS (Cascading Style Sheets; カスケーディングスタイルシート) はウェブコンテンツの見た目や表現を制御するために使います。 詳細で分かり易く使用法が記載されているリファレンスサイトです。

  • CSSフレームワーク Fomantic-UI

    Fomantic UI を使用すると、美しいウェブサイトを素早くデザインできます。人に優しいHTMLを使って、レスポンシブで美しいレイアウトを作成するのに役立つ開発フレームワークです。

  • Can I use ?

    "Can I use "では、デスクトップおよびモバイルのWebブラウザでフロントエンドのWeb技術をサポートするための最新のブラウザサポート表を提供しています。

Ruby / Ruby on Rails

より本格的なプログラミング技術へ。優れた書き味で楽しくコーディングできる Ruby と、ウェブアプリケーションフレームワークとして大人気の Ruby on Rails です。

  • オブジェクト指向スクリプト言語 Ruby

    Rubyとは、オープンソースの動的なプログラミング言語で、 シンプルさと高い生産性を備えています。 エレガントな文法を持ち、自然に読み書きができます。公式サイトです。

  • Ruby on Rails

    Ruby on Railsを学んだら何が作れるか想像してみてください...。 最新のウェブアプリケーションの構築を学ぶのは大変なことです。Ruby on Railsを使えば、もっと簡単に、もっと楽しく学ぶことができます。 Model-View-Controller (MVC) パターンに従ってデータベースを利用したウェブアプリケーションを作成するために必要なすべてのものを含んだウェブアプリケーションフレームワークです。

  • Ruby on Rails ガイド：体系的に Rails を学ぼう

    Ruby on Rails に特化した 1,600 ページ超えの大型リファレンスガイド。プロダクト開発を生産的にする電子書籍や全文検索、法人向けサービスも提供しています。

写真素材

ウェブサイト作成に当たって必要となる様々な写真素材を提供されているサイトです。

  • ぱくたそ

    人気の写真素材を無料でダウンロード「ぱくたそ」写真は33,119枚 使って楽しい、見て楽しい。高解像度で美しい日本の写真が沢山あります。

  • pro.foto

    pro.foto（プロ ドット フォト）では、雑誌・出版業界や広告業界などで活躍しているプロカメラマンが撮影した高品質の写真素材を無料のフリー素材として公開しています。 商業デザインなどの商用利用も可能な高解像度の無料画像がフリー素材として今すぐ無料でダウンロードできます。

  • BEIZ images

    フリー素材のベイツ・イメージズでは、背景や壁紙に使える画像素材を無料でダウンロードできます。高画質＆高解像度の風景やテクスチャーなど、Web、DTP、動画、アプリ、背景等に使える商用利用可能なフリー素材として限りなく自由にご利用いただけます。

  • 足成（あしなり）

    「写真素材 足成」は、全国のアマチュアカメラマンが撮影した写真を、写真素材として無料で提供しています。個人、商用を問わず全て無料でお使い頂け、その上、煩わしいクレジットやリンクの表記、ユーザー登録は一切不要。Web、紙、動画他、あらゆる媒体で写真素材としてのご利用が可能です。

  • Pixabay

    どこでも使える無料のイメージとビデオ. Pixabayは創作に意欲的なコミュニティであり、著作権フリーな画像や映像をお使いいただけます。 海外サイトです。

  • 写真AC

    高品質な写真素材が無料でダウンロードできます。加工や商用利用もOK! 登録やダウンロードが一手間掛かります。

遠隔レッスン

居住地が離れているなど、遠隔でのレッスンご希望の方に使っていただきたいソフトのご紹介です。

  • splashtop – 画面共用ソフト

    自宅にいながら、一緒に画面を見てプログラミング指導が可能とするための画面共用ソフトです。

  • Whereby – ビデオ通話アプリ

    ビデオ通話アプリとして、定評のあるWherebyです。iPhoneにインストールすることで、お顔を見ながらプログラミング指導できます。iPhoneのカメラを画面や書籍に向けることで分からない点の質問等もできます。

開発環境

プログラミングに欠かせない開発環境を整えるためのサイトのご案内です。

  • Airに新しい力をのせて

    MacBookAirは、美しく洗練されたデザイン、心地よく優れた利用者体験を提供するmacOSのもと、プログラミングを始める方への最初の一台として最適な機種です。

  • Atom - 21世紀のプログラマの為のテキストエディタ

    GitHub謹製テキストエディタ。プログラマが心地よくコーディングできるよう、必要な機能が搭載されています。始めての方へは設定不要で初日から使え、熟練者には深部まで調整可能な、21世紀のプログラマの為のテキストエディタです。

  • Firefox - プライバシに配慮した安全性の高いブラウザ

    Firefoxは、Mosaic, Netscape の血を受け継ぐブラウザです。CSSグリッドを確認し易く、丁寧なコメントの開発者ツールが特徴です。

  • Git(ギット) - 分散型バージョン管理システム

    Gitはフリーでオープンソースの分散型バージョン管理システムで、小規模なプロジェクトから大規模なプロジェクトまで、あらゆるものを迅速かつ効率的に処理できるように設計されています。 Git は習得が簡単で、高速なパフォーマンスを発揮します。容易に作成可能なローカルブランチ、便利なステージングエリア、多様なワークフローが特徴です。

  • Pro Git

    Git の 基本操作から応用的な事項まで、500ページに渡って書かれた書籍です。 大部ですが、二章「Gitの基本」だけでも読まれることをお薦めいたします。

  • GitHub(ギットハブ)

    開発者のために作られた GitHub は、あなたの働き方にインスパイアされた開発プラットフォームです。オープンソースからビジネスまで、コードをホストしてレビューしたり、プロジェクトを管理したり、5000万人の開発者と一緒にソフトウェアを構築したりすることができます。

  • Netlify(ネットリファイ)

    静的Webサイトを公開可能なホスティングサービスです。1クリックでHTTPSなど必要なすべてのサービスを無料で利用可能です。

  • heroku(ヘロク)

    Herokuは、開発者がアプリケーションを完全にクラウド上で構築、実行、運用することを可能にするPaaS（Platform as a Service）です。

  • Homebrew - macOS用パッケージマネージャー

    macOS用パッケージマネージャー。開発に必要な様々なプログラムをインストールできます。

  • iTerm2

    macOS用高性能ターミナルアプリ。

便利なツール

ウェブサイト作成に役立つツールのご紹介です。

  • 様々なファビコンを一括生成。favicon generator

    使い方はとってもシンプル。画像を1枚指定するだけ。面倒な設定不要。 favicon.ico、apple-touch-icon*.pngなどを一括生成します。

  • QRコード(二次元バーコード) 作成

    QRコードを無料で作成可能です。

  • ホームページのパーツ作成 - CMAN

    ブラウザ操作だけで、文字や画像のデザインや動きを自動生成することが出来ます。 HTML / CSS / JSが自動で作成されます。 少し古めですが、ご参考に。

  • codic

    codicは、プログラマーのためのネーミング辞書です。新しいcodicでは、翻訳エンジンを搭載しネーミングをジェネレートできるようになりました。

  • deepL

    AIの力を言語に。DeepL（ディープエル）では、テキストをより深く理解し翻訳できる人工知能を開発しています。世界最高レベルの機械翻訳をお試しください。 Macで使えるアプリも提供されているので、ドキュメントを読む際などに重宝します。

  • HTML2slim

    htmlで書かれているサンプルコードをslimに変換する際に便利です。

  • ＨＴＭＬ整形ツール

    このツールは「＜」「＞」マークを元に、HTMLを見やすく整形してくれる単純なツールです。 乱れたhtmlの整形に便利です。
