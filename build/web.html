将棋サークル

対局した棋譜や、会員の方からのさまざまな文章や画像の投稿が可能なサークル用掲示板です。 棋譜データの保存等、データベースを用いた本格的なプログラミングが可能なフレームワーク Ruby on Rails を利用しています。

伊勢湾カップ公式サイト

知多市新舞子を舞台に毎年夏に行われるウィンドサーフィンの祭典。 全国各地から卓越した技術を持つ選手200名が集い、その腕を競います。

株式会社ホームメイド

岐阜市で建築リフォームを営む企業さまのサイトです。 既存のWebサイトがiPhoneに対応しておらず、また大量の施工事例をお客様に分かりやすくお伝えしたいとのことで、事例の絞り込み機能も実装しております。

株式会社ノエル・コンサルサポート

名古屋市の社会保険労務士の方のサイトです。 既存のWebサイトの移行に併せ、iPhoneへの対応、YouTube動画の掲載を行いました。

料金や制作期間について

Webサイトの作成料金や期間は、そのサイトのもつ性格やページ数等、様々な要素があり、一概に申し上げることは困難ですが、トップページと数点の下層ページからなる基本的なWebサイトでございましたら、文章や画像等をご提供頂いた場合、概ね数万円から作成いたしております。作成期間は一ヶ月を目安にしていただければと思います。

サーバーの管理

作成いたしましたWebサイトは、お客様ご自身でお好みのサーバーで公開していただけます。また、当方の推奨するサーバーであれば、月々の更新業務もいたしております。

対応ブラウザ

Mac / iPhone / iPad に対応したWebサイトをお作りいたします。Safari や Google Chrome で閲覧なさってください。

お打ち合わせ

お作りになりたいサイトのイメージなど、お近くの喫茶店で、お打ち合わせをさせていただければと思います。またご予算面でもご相談に乗りますので、お気軽にお問い合わせください。
